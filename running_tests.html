Running tests

执行测试

之前提到的，check需要connectedCheck任务可以执行的一台已经连接的设备。而且还依赖connectedDebugAndroidTest任务，所有将会执行这个任务。这个任务将会做如下操作：

  • Ensure the app and the test app are built (depending on assembleDebug and assembleDebugAndroidTest).
  • Install both apps.
  • Run the tests.
  • Uninstall both apps.

如果有多于一台已连接的设备，所有test会在所有已连接的设备上执行，如果其中一test失败了，那么构建将会失败。
